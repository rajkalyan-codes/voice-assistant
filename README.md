# voice assistant

🎙️ Voice Assistant (Multilingual AI)

🚀 A smart Python-based voice assistant that understands your voice, responds in multiple languages, and performs basic tasks like a mini AI assistant.

🌟 Features

✨ Voice recognition (Speech-to-Text)
🗣️ Text-to-Speech responses
🌍 Multilingual support (English / Hindi / Punjabi)
⏰ Get current time by voice command
💬 Smart conversational replies
🔁 Continuous listening mode
❌ Exit with simple voice command

🧠 Tech Stack
Python 🐍
SpeechRecognition 🎤
pyttsx3 🔊
Google Translate API 🌐
PyAudio 🎧
📸 Demo

🎤 “Hello” → Assistant replies in selected language
🎤 “What is the time” → Gives current time
🎤 “Exit” → Stops assistant

⚙️ Installation
1️⃣ Clone the repository
git clone https://github.com/your-username/voice-assistant.git
cd voice-assistant
2️⃣ Install dependencies
pip install SpeechRecognition pyttsx3 pyaudio googletrans==4.0.0-rc1

👉 If PyAudio error occurs (Windows):

pip install pipwin
pipwin install pyaudio
▶️ Run the Project
python main.py
🧑‍💻 How It Works
Choose your language 🎯
Speak into microphone 🎤
Assistant listens & understands 🧠
It replies in your selected language 🔊
💡 Example Commands
Voice Input	Response
Hello	Greetings
What is the time	Current time
Exit / Stop	Closes assistant
